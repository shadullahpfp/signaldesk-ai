# SignalDesk AI

SignalDesk AI is an AI-powered customer workflow and intelligence platform.

This project is built as a flagship portfolio project demonstrating:
- FastAPI Backend Architecture
- PostgreSQL / Supabase
- JWT Authentication
- Next.js 15 Frontend
- AI Integration (OpenAI)
